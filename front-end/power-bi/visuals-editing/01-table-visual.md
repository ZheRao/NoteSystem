## Turn off Auto-size column width

1. Click the **Table visual**
2. Go to **Format visual (paint roller)**
3. Expand **Column headers**
4. Toggle **Auto-size column width → Off**

This is the *only* official control that stops Power BI from stretching columns to fill the table.

✅ After this:
- Column widths become **manual**
- Power BI stops re-calculating widths when data changes
